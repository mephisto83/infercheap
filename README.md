# InferCheap

KV-cache-free inference engine for memory-efficient LLM inference.

## Features
- 40-60% memory reduction vs standard inference
- Configurable quality/speed tradeoffs
- Multiple model architecture support
- Latency monitoring and optimization
- React + Tailwind SaaS webapp with Firebase Auth

## Quick Start
1. Clone this repo
2. Install: `pip install -e .`
3. Run: `infercheap --help`

## Web App
The `webapp/` directory contains a full SaaS frontend built with React + Tailwind CSS.
